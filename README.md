### How to develop locally

On my macbook, the Python in the system is no good. Use a conda env for good Python instead.

- Step 0: nvm use 24
- Step 1: `npm install --legacy-peer-deps`
- Step 2: `npm run build`
- Step 3: `gatsby develop`
