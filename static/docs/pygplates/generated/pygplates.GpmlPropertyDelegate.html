pygplates.GpmlPropertyDelegate

class pygplates.GpmlPropertyDelegate

Bases: PropertyValue

A property value that represents a reference, or delegation, to a property in another feature.

A GpmlPropertyDelegate can also be pickled.

Added in version 0.21.

Changed in version 0.42: Added pickle support.

__init__(feature_id, property_name, property_type)

Create a reference, or delegation, to a property in another feature.

Parameters:

  • feature_id (FeatureId) – the referenced feature

  • property_name (PropertyName) – the name of the referenced property

  • property_type (a class object of a property type (derived from PropertyValue) except Enumeration and time-dependent wrappers GpmlConstantValue, GpmlIrregularSampling and GpmlPiecewiseAggregation.) – the type of the referenced property

property_delegate = pygplates.GpmlPropertyDelegate(
    referenced_feature.get_feature_id(),
    pygplates.PropertyName.gpml_center_line_of,
    pygplates.GmlLineString)
